# Fitur Riwayat Order

Kembali ke summary: [app-summary.md](../app-summary.md)

## Tujuan

Fitur ini menyediakan daftar dan detail order yang sudah tercatat di aplikasi dalam konteks operasional kasir.

## Cakupan Fitur

- daftar riwayat order
- filter status order
- konteks shift aktif / hari ini
- detail item order
- detail pembayaran
- status fulfillment
- aksi void untuk order tertentu
- aksi pelunasan dari riwayat
- refund / pembatalan pada kondisi tertentu

## Peran Dalam Aplikasi

- menjadi pusat monitoring order yang sudah dibuat
- membantu kasir meninjau, melanjutkan, atau membatalkan order sesuai kebutuhan operasional

## Area yang Terkait

- tab Riwayat
- pembayaran
- shift
- status order POS

## Catatan

- Riwayat order di aplikasi ini lebih dekat ke kebutuhan kasir daripada sekadar daftar transaksi generik.
